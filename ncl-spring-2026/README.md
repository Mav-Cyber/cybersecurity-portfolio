## 📖 Case Studies

### Log Analysis: Transaction Data Investigation
**Key Lesson**: Standardized identifiers are more reliable than text searching

Demonstrated advanced CSV analysis using awk, showing how field-based filtering and conditional aggregation outperform text search for structured data.

→ [Full Case Study](../ctf-writeups/ncl-mcc-easy-challenge.md)

**Key Lesson**: Individual events are meaningful only in context; timeline analysis reveals attacker intent
